⚠️ Proxy kurulumu gerekiyor

Tarayıcı güvenlik politikaları nedeniyle Türk haber sitelerine doğrudan bağlanmak mümkün olmuyor. Aşağıdaki ücretsiz Google Apps Script proxy'ini kurarak haberleri çekebilirsiniz:

  1. script.google.com adresine git → Yeni proje oluştur
  2. Aşağıdaki kodu yapıştır ve kaydet
  3. Deploy → New deployment → Web app seç → "Execute as: Me", "Who has access: Anyone" yap → Deploy et
  4. Aldığın URL'yi aşağıya yapıştır ve Kaydet'e bas
function doGet(e) {
  var url = e.parameter.url;
  if (!url) return ContentService.createTextOutput(JSON.stringify({error:"missing url"})).setMimeType(ContentService.MimeType.JSON);
  try {
    var response = UrlFetchApp.fetch(url, {muteHttpExceptions: true, followRedirects: true});
    return ContentService.createTextOutput(response.getContentText()).setMimeType(ContentService.MimeType.TEXT);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({error: err.toString()})).setMimeType(ContentService.MimeType.JSON);
  }
}
