⚡ Proxy Kurulumu

Türk haber sitelerini tarayıcıdan doğrudan çekmek mümkün olmuyor. Aşağıdaki ücretsiz Google Apps Script proxy'ini bir kez kurarak tüm haberleri anında alabilirsiniz.

  1. script.google.com → Yeni proje oluştur
  2. Aşağıdaki kodu yapıştır ve kaydet (Ctrl+S)
  3. Deploy → New deployment → Web app → Execute as: Me · Access: Anyone → Deploy
  4. Çıkan URL'yi aşağıya yapıştır ve Kaydet'e bas
function doGet(e) {
  var url = e.parameter.url;
  if (!url) return ContentService.createTextOutput("missing url");
  var res = UrlFetchApp.fetch(url, {muteHttpExceptions:true, followRedirects:true});
  return ContentService.createTextOutput(res.getContentText())
    .setMimeType(ContentService.MimeType.TEXT);
}
